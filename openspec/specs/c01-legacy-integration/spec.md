## Requirements
### Requirement: 舊有專案資產存放路徑
The system SHALL store all legacy AIoT Final Project documents within the `docs/legacy/` directory to serve as reference materials for new architecture designs.

#### Scenario: 存取舊版本架構簡報
- **WHEN** 開發者想要了解先前的「Smart Posture AIoT System」架構
- **THEN** 他們可以在本專案的 `docs/legacy/Smart_Posture_AIoT_System.pptx` 與相關 pdf 中直接開啟閱讀。
